# Taylor Tour Tracker

Extracts and stores data from the ACCC TayTaywatch charts at https://www.accc.gov.au/consumers/petrol-and-taytay/petrol-price-cycles-in-the-5-largest-cities.

Analysis is basic and outputted on [GitHub Pages](https://MMF-27.github.io/taylor-tour-tracker/).

Raw data is saved into petrol_prices.csv
